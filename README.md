# Ethio Keyboard 

A modern, responsive Ethiopian Geez Keyboard Web Application built using **HTML5, CSS3, and Vanilla JavaScript**. The application allows users to type Ethiopian languages using English transliteration, similar to the Fyn Geez Keyboard.

---

## Features

- 🇪🇹 Real-time Geez transliteration
-  Interactive virtual keyboard
-  Multiple language modes
  - Amharic
  - Afaan Oromo
  - Tigrinya
  - Geez
  - English
-  Light & Dark Theme
-  Multiple color themes
-  Auto Save using LocalStorage
-  Copy, Paste, Undo, Redo
-  Import and Export TXT files
-  Export as PDF and DOC
-  Print support
-  Ethiopic character search
-  Emoji picker
-  Font selector
-  Adjustable font size
-  Favorite characters
-  Ethiopic alphabet reference
-  Fully responsive design
-  Accessibility support
-  Fast, lightweight, and works offline
